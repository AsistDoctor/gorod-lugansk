# Техническое задание
## Редизайн сайта gorod-lugansk.ru

---

### 1. Общая информация

**Заказчик:** Муниципальное образование городской округ город Луганск ЛНР  
**Текущий сайт:** https://gorod-lugansk.ru/  
**Тип проекта:** Редизайн и разработка нового сайта с миграцией контента

---

### 2. Цели проекта

- Создать современный, удобный и адаптивный сайт
- Улучшить пользовательский опыт и навигацию
- Обеспечить удобное управление контентом для редакторов
- Соответствовать требованиям доступности для госсайтов

---

### 3. Технический стек

| Компонент | Технология |
|-----------|------------|
| Фронтенд | Nuxt 3 (Vue 3) |
| Бэкенд/CMS | Strapi |
| База данных | PostgreSQL |
| Стилизация | Tailwind CSS |
| Деплой (фронт) | Vercel / Netlify |
| Деплой (бэкенд) | Railway / Render / VPS |

---

### 4. Функциональные требования

#### 4.1 Структура сайта (разделы)

**Главное меню:**
- Наш город
- Структура ОМС
- Глава городского округа
- Совет депутатов
- Администрация

**Дополнительное меню:**
- Распоряжения
- Постановления
- Проекты

**Служебные элементы:**
- Поиск по сайту
- Версия для слабовидящих
- Ссылки на социальные сети

#### 4.2 Главная страница

- Шапка с логотипом и названием
- Главное меню навигации
- Блок "Тема дня" (важная информация)
- Лента новостей с датами
- Быстрые ссылки на ключевые разделы
- Контакты / телефоны экстренных служб
- Футер с дополнительной навигацией

#### 4.3 Страница новости

- Заголовок
- Дата публикации
- Текст новости (с поддержкой форматирования, изображений)
- Категория/тег
- Кнопки "поделиться"
- Связанные новости

#### 4.4 Раздел документов

- Фильтрация по типу (распоряжения, постановления, проекты)
- Фильтрация по дате
- Поиск по названию
- Скачивание файлов (PDF)

#### 4.5 Информационные страницы

- О городе
- Структура администрации
- Контакты
- Страницы подразделений

---

### 5. Нефункциональные требования

#### 5.1 Дизайн
- Современный, чистый интерфейс
- Нейтральная цветовая палитра (легко адаптировать под брендбук)
- Читаемая типографика
- Достаточные отступы и воздух

#### 5.2 Адаптивность
- Корректное отображение на десктопе, планшете, мобильных
- Mobile-first подход

#### 5.3 Доступность
- Версия для слабовидящих (увеличение шрифта, контрастные темы)
- Семантическая вёрстка
- Alt-тексты для изображений
- Навигация с клавиатуры

#### 5.4 Производительность
- Оптимизация изображений
- Ленивая загрузка контента
- Кэширование

#### 5.5 SEO
- SSR (Server-Side Rendering) через Nuxt
- Мета-теги для страниц
- Sitemap.xml
- Robots.txt

---

### 6. Административная панель (Strapi)

#### 6.1 Роли пользователей
- **Администратор** — полный доступ
- **Редактор** — создание/редактирование новостей и документов

#### 6.2 Типы контента
- Новости (заголовок, текст, изображение, дата, категория)
- Документы (название, тип, файл, дата)
- Страницы (произвольный контент)
- Категории
- Контакты / телефоны

---

### 7. Миграция контента

- Перенос существующих новостей
- Перенос документов (PDF файлы)
- Перенос структуры разделов
- *Выполняется после согласования с заказчиком*
